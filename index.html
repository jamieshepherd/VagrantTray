VagrantTray

A simple to use tray menu manager for Vagrant on Windows. Access all of the popular vagrant commands through a simple GUI.

View project onGitHub

Welcome to VagrantTray.

VagrantTray is an easy to use Vagrant manager for Windows. It features a simple tray based GUI menu and keeps out of your way while you work on your projects with Vagrant. It's completely free and open-source, I'd love for you to use it or contribute towards making it better!

Easy to get started

Just go to Add New and select a Vagrantfile. That's it.

Manage your box

Some of the more popular vagrant commands in a simple menu including vagrant up, halt, reload and ssh. VagrantTray keeps the command line out of your way for the most part (unless you're using SSH) so you can quickly load your box and get on with your work.

Lightweight

VagrantTray is really small, so there's really no downside to trying it out and having it around. It can (and should) become an integral part of your workflow.

Authors and Contributors

VagrantTray was made by @jamieshepherd after being inspired by the work of Vagrant Manager for Mac OS X. You can check them out here. If you'd like to contribute, just fork away. This was mainly a learning project for me so I'm sure there is still a ton of work to be done if you're willing.
